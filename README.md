MPS-Qt5
=======

DICOM Image Server

This is the first version of my DICOM Server

It has support for the following services:
1.- C-ECHO (SCP/SCU)
2.- S-STORE (SCP/SCU)
3.- C-FIND (SCP/SCU)
4.- C-MOVE (SCP/SCU)

For the develop of this first version I use the following technologies:
1.- Qt5
2.- MongoDB
3.- C++11 Standard
4.- DCMTK 3.6
5.- KDevelop 4.5 (as IDE)
